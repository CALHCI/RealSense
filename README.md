# RealSense
A framework to gather data from Intel RealSense devices


## Installation Guidelines (type in terminal):

camVision.py, depth.py, :
-pip install numpy
-pip install pyrealsense2
-pip install opencv-python

objects.py:
-pip install opencv-python
-pip install numpy
